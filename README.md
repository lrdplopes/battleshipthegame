## 📁 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/lrdplopes/battleshipthegame

    #Acessar o repositório
    $ cd battleshipthegame

    #Instalar suas dependências
    $yarn install

    #Iniciar o projeto
    $yarn start

```
